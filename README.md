
# Go-shopping
E-Commerc app
## منصة التجارة الإلكترونية لبيع الملابس (تطبيق متكامل متعدد الصلاحيات)
# تطبيق احترافي متكامل للهواتف الذكية مبني باستخدام إطار العمل Flutter، مخصص لإدارة عمليات البيع والشراء والشحن للملابس. يتميز التطبيق بالقدرة على إدارة ثلاثة أدوار مختلفة للمستخدمين (العميل، التاجر، ومندوب التوصيل) عبر نظام متزامن ولحظي.
